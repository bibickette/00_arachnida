créer un venv : `python3 -m venv .venv`
lancer mon env python `source .venv/bin/activate`
    et quitter : `deactivate`

installer des packages : activate et `pip install package_name` 
lister les packages installés : `pip list`
mettre a jour : `pip install --upgrade package_name`
uninstall : `pip uninstall package_name`
voir les dependances dun packet : `pip show package_name` 

exporter les dependances dans un requirements.txt (cest comme le package.json) : `pip freeze > requirements.txt`
installer les dependances depuis un requirements.txt : `pip install -r requirements.txt`

verifier les mise a jours de secu : `pip list --outdated`

Ajouter le shebang (comme mentionné ci-dessus). #!/usr/bin/env python3
Changer les permissions du fichier pour le rendre exécutable à l’aide de la commande chmod :
Fenêtre de terminal
chmod +x mon_script.py

Exécuter le script directement depuis le terminal :
Fenêtre de terminal
./mon_script.py
ou
python3 mon_script.py

Ouvrir le python interactif : dans le .venv activé, taper python, la on peut ecrire du code




Script : fichier texte interprété par un langage (Python, Bash, etc.)
Binaire : programme compilé en code machine (C, C++, Go…)
